# Information Security Baseline

## Account and Access
All employee accounts must use MFA.
Passwords must be at least 12 characters and rotated every 180 days.

## Data Classification
Data is classified into Public, Internal, Confidential, and Restricted.
Confidential or higher data must not be shared through personal email.

## Endpoint Security
Company-managed laptops must enable full-disk encryption.
OS security patches must be applied within 7 days of release for critical updates.

## Incident Reporting
Security incidents must be reported within 1 hour after discovery.
The security team must provide an initial impact assessment within 24 hours.
