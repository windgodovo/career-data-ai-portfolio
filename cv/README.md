# 岗位定制 CV 索引

这个目录用于存放按岗位方向定制的简历草稿。每份 CV 都以当前作品集项目为核心证据，避免一份简历投所有岗位。

## 文件说明

| 文件 | 适合岗位 |
|---|---|
| `cv-game-data-analyst.md` | 游戏数据分析、游戏 DA、版本分析、数值/战斗策划数据支持 |
| `cv-ai-rag-data-analyst.md` | AI 数据应用、RAG 应用分析、LLM 产品数据、AI Solutions Analyst |
| `cv-data-engineer-bi.md` | 数据开发、数据工程、BI 数仓、数据实施 |
| `cv-risk-strategy-analytics.md` | 风控策略、Trust & Safety Analytics、内容安全策略、银行风控分析 |
| `cv-reporting-automation-bi.md` | 报表自动化、运营分析、BI 自动化、Fiverr/Upwork 自动化服务 |

## 使用方式

1. 先替换姓名、邮箱、LinkedIn、GitHub、所在地等个人信息。
2. 根据 JD 删除不相关项目，只保留 2-3 个最贴近岗位的项目。
3. 把“项目链接”替换为 GitHub 仓库中的具体项目路径。
4. 如果投英文远程岗位，可在此基础上再翻译成英文版。
