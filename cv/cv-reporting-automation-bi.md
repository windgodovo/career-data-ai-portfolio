## 重点项目

### Excel AI 自动报表助手

项目链接：`03-excel-ai-auto-report-agent`

- 构建自动报表 Agent，从 CSV / Excel 读取销售数据，完成字段校验和数据清洗。
- 将无效行分离到校验日志，避免污染业务指标。
- 自动计算整体、区域、月份等维度 KPI。
- 使用 LLM 生成业务摘要；未配置 API Key 时使用本地兜底摘要，保证可运行。
- 输出 CSV、TXT、JSON 报告，并支持可选 SMTP 邮件发送。
- 提供 cron 示例，适合部署为每日自动报表任务。
