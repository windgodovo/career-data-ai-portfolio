# 指标字典

## 1. 活跃与留存

| 指标 | 口径 | 用途 |
|---|---|---|
| DAU | 当日登录玩家数 | 判断版本活跃 |
| 新增玩家 | 首次登录玩家数 | 判断拉新效果 |
| 次留 / 7 留 | 新增玩家第 2 / 7 天回访比例 | 判断新手体验 |
| 回流玩家 | N 天未登录后再次登录 | 判断赛季回流 |
| 流失玩家 | 连续 N 天未登录 | 流失监测 |

## 2. 付费指标

| 指标 | 口径 | 用途 |
|---|---|---|
| 付费人数 | 当日成功付费玩家数 | 付费规模 |
| 付费率 | 付费人数 / DAU | 转化能力 |
| ARPU | 收入 / DAU | 单活跃收入 |
| ARPPU | 收入 / 付费人数 | 付费用户价值 |
| LTV | 玩家生命周期累计收入 | 商业化评估 |

## 3. 对局指标

| 指标 | 口径 | 用途 |
|---|---|---|
| 开局数 | `match_start` 数量 | 对局活跃 |
| 完局率 | `match_end` / `match_start` | 稳定性与体验 |
| 平均对局时长 | `duration_sec` 平均值 | 节奏健康度 |
| 排位占比 | 排位对局 / 总对局 | 竞技参与度 |
| 平均回合数 | 完局回合平均值 | 节奏变化 |

## 4. 阵容与平衡指标

| 指标 | 口径 | 用途 |
|---|---|---|
| 出场率 | 使用该阵容的对局玩家数 / 总对局玩家数 | meta 热度 |
| 前四率 | 排名 1-4 的次数 / 使用次数 | 稳定上分能力 |
| 登顶率 | 排名第 1 次数 / 使用次数 | 极限强度 |
| 平均名次 | 使用该阵容的平均排名 | 综合强度 |
| 阵容集中度 | Top N 阵容出场率合计 | meta 是否单一 |

## 5. 经济系统指标

| 指标 | 口径 | 用途 |
|---|---|---|
| 平均升级回合 | 达到 6/7/8/9 人口的平均回合 | 运营节奏 |
| D 牌频率 | 每阶段 roll 次数 | 阵容成型压力 |
| 金币留存 | 回合结束金币余额 | 经济健康度 |
| 连胜/连败收益 | 连胜连败路径收益 | 经济策略平衡 |

## 6. 告警指标

- DAU 日环比跌幅超过阈值。
- 付费率或 ARPU 异常下跌。
- 完局率异常下降。
- 单一阵容出场率过高。
- 某棋子/羁绊前四率或登顶率异常偏高。
- 补丁后核心阵容前四率异常暴跌。
