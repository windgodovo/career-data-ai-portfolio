# 金铲铲 DA 指标体系

## 1. 日常经营指标

| 指标 | 口径 | 用途 |
|---|---|---|
| DAU | 当日活跃账号数 | 判断整体活跃规模 |
| 新增用户 | 当日首次进入游戏账号数 | 观察拉新效果 |
| 次日留存 | D0 新增用户在 D1 回访比例 | 判断新手体验 |
| 7 日留存 | D0 新增用户在 D7 回访比例 | 判断中期留存 |
| 付费率 | 当日付费用户 / 当日活跃用户 | 判断商业转化 |
| ARPU | 当日收入 / 当日活跃用户 | 收入效率 |
| ARPPU | 当日收入 / 当日付费用户 | 付费深度 |
| 对局数 | 当日完成对局数量 | 玩法参与度 |
| 人均对局 | 对局数 / DAU | 核心玩法粘性 |

## 2. 对局与玩法指标

| 指标 | 口径 | 用途 |
|---|---|---|
| 平均对局时长 | 完成对局总时长 / 对局数 | 判断节奏是否过长或过短 |
| 前四率 | 阵容进入前四次数 / 阵容出场次数 | 自走棋强度指标 |
| 登顶率 | 阵容第一名次数 / 阵容出场次数 | 判断上限强度 |
| 出场率 | 使用某阵容的对局数 / 总对局数 | 判断 meta 热度 |
| 装备组合胜率 | 某棋子+装备组合进入前四比例 | 识别装备强度 |
| 羁绊激活率 | 激活某羁绊的玩家局数 / 总玩家局数 | 判断羁绊使用情况 |
| 平均淘汰回合 | 玩家被淘汰的平均回合 | 判断经济和节奏问题 |

## 3. 版本 / 补丁分析指标

| 指标 | 分析方法 |
|---|---|
| 阵容胜率变化 | 对比补丁前后同一阵容前四率、登顶率、出场率 |
| 棋子强度变化 | 对比补丁前后携带该棋子的阵容表现 |
| 羁绊平衡 | 对比不同羁绊激活层级下的排名分布 |
| 经济系统影响 | 对比平均等级、金币留存、升人口时间、D 牌次数 |
| 用户体验影响 | 对比对局时长、投降率、流失率、局后负反馈 |

## 4. 告警指标

| 告警项 | 条件示例 |
|---|---|
| DAU 异常 | 今日 DAU 低于近 7 日均值 15% |
| 支付异常 | 付费率或 ARPU 日环比下降超过 20% |
| 对局异常 | 对局完成率下降或异常中断率升高 |
| 阵容失衡 | 某阵容前四率 > 65% 且出场率 > 15% |
| 补丁风险 | 补丁后核心阵容前四率暴跌超过 20 个百分点 |
