# 阵容 Meta 与平衡性分析

## 1. 自走棋分析重点

金铲铲 / TFT 的 DA 不能只看单个棋子胜率，因为强度来自组合：

```text
棋子强度 + 羁绊联动 + 装备适配 + 经济节奏 + 卡池概率 + 对手环境
```

## 2. 核心对象

| 对象 | 分析指标 |
|---|---|
| 阵容 | 出场率、前四率、登顶率、平均名次、成型回合 |
| 棋子 | 登场率、三星率、携带装备、进入前四贡献 |
| 羁绊 | 激活率、激活阶段、组合胜率 |
| 装备 | 携带率、绑定棋子、前四率、登顶率 |
| 经济路径 | 快 8、慢 D、连胜、连败、赌狗阵容 |

## 3. Meta 迁移分析

分析玩家从补丁前阵容迁移到补丁后阵容：

```text
旧版本阵容 A 玩家
  ↓
补丁后继续玩 A / 迁移到 B / 迁移到 C / 流失
```

可回答：

- 被削弱阵容的玩家去了哪里。
- 新强势阵容是否来自原本某类玩家群体。
- 高分段和低分段 meta 是否不同。
- 国内版和海外版是否出现不同理解。

## 4. 平衡判断框架

| 情况 | 判断 |
|---|---|
| 高出场 + 高前四 + 高登顶 | 大概率过强 |
| 低出场 + 高前四 | 可能是高手专属或冷门强势 |
| 高出场 + 低前四 | 热门但不强，可能是娱乐或跟风 |
| 低出场 + 低前四 | 需要增强或机制不吸引 |

## 5. 策划建议表达

不要只说“胜率高”，要说：

> 该阵容在大师以上分段出场率从 8% 提升到 21%，前四率 61%，登顶率 18%，且核心棋子携带指定装备后的登顶率达到 26%。主要强度来自 7 人口成型过早和两件装备收益过高，建议优先削弱核心棋子的技能数值或装备收益，而不是直接削羁绊整体。
