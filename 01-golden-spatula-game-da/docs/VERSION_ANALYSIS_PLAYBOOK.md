# 版本 / 补丁分析 Playbook

## 1. 分析目标

每次赛季或补丁上线后，DA 需要快速回答：

- 活跃和付费是否受影响。
- 对局节奏是否变化。
- 强势阵容是否被削弱。
- 弱势阵容是否被扶起来。
- 是否出现新的超模棋子、羁绊或装备组合。
- 国内版和海外版的 meta 是否一致。

## 2. 分析步骤

1. 确定分析窗口：补丁前 7 天 vs 补丁后 1/3/7 天。
2. 分层维度：地区、段位、模式、平台、新老玩家。
3. 看整体盘面：DAU、对局数、付费、完局率、时长。
4. 看平衡指标：阵容出场率、前四率、登顶率、平均名次。
5. 看结构变化：棋子、羁绊、装备、经济节奏。
6. 看玩家迁移：老阵容玩家转向了哪些新阵容。
7. 输出策划建议：削弱、增强、观察、继续灰度。

## 4. 常见结论类型

| 现象 | 可能原因 | 建议 |
|---|---|---|
| 阵容出场率高且登顶率高 | 强度过高 | 削弱核心棋子/羁绊/装备 |
| 出场率低但前四率高 | 冷门强势或理解门槛高 | 观察高分段，谨慎调整 |
| 出场率高但胜率低 | 热门但弱，可能是娱乐阵容 | 不急于削弱 |
| 前四率暴跌 | 核心棋子削弱、装备变动、经济节奏变化 | 拆解归因 |
| 单阵容过度集中 | meta 单一 | 增强克制阵容或削弱核心路径 |
