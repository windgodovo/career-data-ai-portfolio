# Tableau / Excel 看板设计

## 1. 制作人总览看板

核心卡片：

- DAU
- 收入
- 付费率
- 对局数
- 完局率
- 次留 / 7 留
- 高风险平衡告警数

趋势图：

- DAU / 收入日趋势
- 对局数 / 完局率趋势
- 版本上线前后对比

## 2. 版本分析看板

维度筛选：

- 版本号
- 赛季
- 地区：国内 / 海外
- 模式
- 段位
- 平台

图表：

- 版本前后 DAU、对局数、收入变化。
- 阵容出场率 / 前四率散点图。
- 棋子登场率和胜率 Top 榜。
- 羁绊组合强度矩阵。
- 装备组合强度排行。

## 3. 阵容平衡看板

重点图表：

- 阵容出场率 vs 前四率象限图。
- Top 阵容登顶率趋势。
- 不同段位阵容差异。
- 国内 vs 海外 meta 对比。
- 阵容集中度趋势。

## 4. 告警规则

| 告警 | 条件 |
|---|---|
| 单阵容过强 | 出场率 > 15% 且前四率 > 58% |
| 阵容暴跌 | 前四率较补丁前下降超过 8pct |
| 完局率异常 | 完局率日环比下降超过 5pct |
| 付费异常 | ARPU 日环比下降超过 10% |
| meta 集中 | Top5 阵容出场率合计超过 50% |
