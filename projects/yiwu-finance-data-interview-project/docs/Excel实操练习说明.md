# Excel 实操练习说明

这个项目主要用于准备义乌财务数据岗位的 Excel 实操。

## 练习目标

你需要用 Excel 从原始订单、支付、退款三张表中，完成：

1. 数据清洗
2. 支付流水匹配
3. 财务日报汇总
4. 对账异常识别
5. 清洗日志记录

---

## 第一步：保留原始数据

打开 `excel/yiwu_finance_demo.xlsx` 后，先不要直接修改原始 Sheet。

建议保留：

- `order_raw`
- `pay_raw`
- `refund_raw`

然后新增：

- `clean_orders`
- `daily_finance_report`
- `reconciliation_alerts`
- `cleaning_log`

面试时可以说：

> 我习惯保留原始数据不动，所有处理都在新 Sheet 中完成，方便后续追溯。

---

## 第二步：清洗订单表

重点处理：

- 删除空行
- 删除重复订单
- 日期格式统一
- 金额字段转数值
- 省份错别字修正
- 文本字段去除前后空格

### 金额清洗公式

```excel
=VALUE(SUBSTITUTE(SUBSTITUTE(A2,"¥",""),",",""))
```

### 去除空格和不可见字符

```excel
=TRIM(CLEAN(A2))
```

### 省份修正

```excel
=SUBSTITUTE(SUBSTITUTE(A2,"浙冮","浙江"),"廣东","广东")
```

---

## 第三步：匹配支付流水

用 `订单号` 将订单表和支付流水表匹配。

建议使用：

```excel
=XLOOKUP(A2,pay_raw!B:B,pay_raw!D:D,"")
```

可以分别匹配：

- 实收金额
- 平台手续费
- 支付渠道

---

## 第四步：计算应收和差异

建议字段：

- 订单金额
- 运费
- 退款金额
- 平台手续费
- 应收金额
- 实收金额
- 差异金额
- 异常原因

计算逻辑：

```text
应收金额 = 订单金额 + 运费 - 退款金额 - 平台手续费
差异金额 = 应收金额 - 实收金额
```

异常判断公式示例：

```excel
=IF(ABS(应收金额-实收金额)<=0.01,"正常","异常")
```

---

## 第五步：做财务日报

可以用透视表或 `SUMIFS` 汇总。

建议维度：

- 日期
- 店铺

建议指标：

- GMV
- 订单量
- 退款金额
- 平台手续费
- 实收金额
- 净收入

净收入：

```text
净收入 = 实收金额 - 退款金额 - 平台手续费
```

---

## 面试表达

> 我会先保留原始数据，然后在新 Sheet 里完成清洗、匹配和汇总。  
> 对异常数据，我会单独输出异常表，并记录清洗日志。这样别人可以按订单号追溯回原始记录。
