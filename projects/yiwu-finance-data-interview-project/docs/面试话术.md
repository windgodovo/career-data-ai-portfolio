# 面试话术

## 1 分钟自我介绍

您好，我叫罗云峰，之前主要围绕数据分析和自动化报表方向做项目，常用工具是 Excel、SQL 和 Python。

针对这个岗位，我重点准备了订单数据清洗、财务对账和日报汇总相关内容。比如我会用 Excel 处理重复行、日期格式、金额格式、文本字段不统一等问题，也会用 XLOOKUP、SUMIFS、透视表来做订单金额、实收金额、退款金额的核对和汇总。

所以我觉得自己比较适合财务数据处理、订单对账和运营日报这类工作。

## 项目介绍话术

我准备了一个义乌电商订单财务对账项目，模拟订单、支付流水和退款三张表。

我先用 Excel 处理重复行、日期格式、金额格式、省份错别字等问题，然后用 XLOOKUP、SUMIFS、COUNTIFS、IF 和透视表完成日报汇总和订单对账。

同时，我也用 Pandas 写了自动化脚本，可以自动输出财务日报、异常明细表和清洗日志。

这个项目重点不是单纯做图表，而是保证数据准确、过程可追溯、异常能定位，比较贴合财务数据岗位的日常工作。

---

## 如果问：你怎么保证数据准确？

我会保留原始数据 Sheet 不动，所有处理都在新 Sheet 完成。

清洗时会记录 cleaning_log，比如删除了哪些重复行、修正了哪些异常字段。

对账时会单独输出异常表，比如缺支付流水、金额不一致、有退款但未扣减等情况。这样后续财务或运营同事可以按订单号追溯回原始记录。

---

## 如果问：Excel 和 Python 你怎么选择？

如果是临时分析或者面试实操，我会优先用 Excel，因为它直观、方便检查，也适合和财务同事协作。

如果是每天都要重复做的日报、周报，我会用 Pandas 写成脚本，把清洗、汇总、异常输出自动化。这样可以减少人工复制粘贴和公式拖拽导致的错误。

---

## 如果问：你会哪些 Excel 函数？

我常用的是 XLOOKUP、VLOOKUP、SUMIFS、COUNTIFS、IF、IFERROR、LEFT、MID、RIGHT、TEXT、TRIM、CLEAN、SUBSTITUTE。

财务数据里面我用得比较多的是 SUMIFS 做多条件汇总，XLOOKUP 做订单和支付流水匹配，IF 判断异常状态，SUBSTITUTE 和 VALUE 处理金额格式。
