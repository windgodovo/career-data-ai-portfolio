## 统一能力地图

### 1. 数据基础

- SQL：CTE、窗口函数、聚合、JOIN、指标口径开发。
- 数据建模：事实表、维度表、宽表、指标层、数据质量表。
- 数据分层：ODS、DWD、DWS、ADS、FEATURE、SERVING、MONITOR。

### 2. 数据平台

- DuckDB / PostgreSQL：本地可复现数据仓库与分析环境。
- Spark / PySpark：作为大规模批处理和滑动窗口特征计算的扩展方向。
- Kafka：作为实时事件采集、交易流、状态流的扩展方向。
- Airflow / Dagster：作为生产环境任务编排扩展方向。

### 3. 数据治理

- 数据标准：字段命名、枚举值、必填字段、主外键关系。
- 数据质量：空值、重复、金额异常、时间异常、分布漂移。
- 数据血缘：从原始数据到指标结果的表级和字段级映射。
- 数据安全：敏感字段脱敏、权限分层、审计记录。

### 4. 业务分析重点

- 游戏：埋点设计、版本分析、阵容 meta 迁移、羁绊平衡、LTV 与流失预警。
- 银行：客户分层、交易异常、账户风险、指标监控、监管口径。
- 直播平台：违规风险识别、举报与审核分析、团伙识别、策略迭代、误杀漏过监控。
- 新能源：站点运营、用户偏好、推荐排序、服务稳定性。
- 企业运营：报表自动化、业务复盘、指标解释、自动化报告。
